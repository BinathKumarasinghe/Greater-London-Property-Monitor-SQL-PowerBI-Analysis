Greater London Property Price Monitor — End-to-End Data Analysis


📌 Overview

A complete data analysis pipeline for the UK Price Paid Dataset,
focused on Greater London residential property transactions from 2020 to 2023.
Covers SQL database design, data cleaning, and interactive Power BI dashboards
built for property investors, estate agents, and policymakers.


🗂️ Project Structure

- Database: Microsoft SQL Server with yearly tables (pp-2020 to pp-2023)
- Tables: Separate tables per year, merged for unified Power BI analysis
- Dashboards: 4-page interactive Power BI report with filters & navigation


✨ Key Features

- NULL value handling across SAON, PAON, Locality & Street columns
- Year-by-year and quarterly transaction trend analysis
- Property type breakdowns: Flat, Detached, Semi-Detached, Terraced & Other
- Geographic distribution across 32 London boroughs and 115 counties
- Power BI visuals: pie charts, line charts, bubble charts, treemaps & decomposition trees


💡 Insights Uncovered

- Flats (27.21%) and Semi-Detached (26.44%) are the most transacted property types
- 2021 saw the peak in transaction volume at over 1.2M transactions
- Average property price rose steadily from 2020 to 2023, with the biggest jump in 2022
- Q2 and Q4 consistently show the highest total prices paid — seasonal market patterns
- Freehold properties command higher average prices compared to Leasehold across all years


🛠️ Tools & Technologies

- Microsoft SQL Server
- Power BI Desktop
- SQL (DDL, DML, Data Cleaning, Aggregations)
- UK Government Price Paid Dataset (1995–Present)
